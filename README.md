# Toggle Sprint Display 🏃
Adds a simple HUD element which shows whether you're sprinting or not. Helps you spot when toggle sprint isn't actually toggled.

![UI Example](https://cdn.modrinth.com/data/shVMaevq/images/2faa519f25ede26ca4a71c4e91d0be5cc87e2d41.png)

## Extra Features 🌟
You can customise the HUD element however you want!
- Pick whether it shows sprinting, sneaking and/or swimming
- Choose between short words or emojis
- Change the alignment, position and size
- Show a small paper doll of the player on-screen
